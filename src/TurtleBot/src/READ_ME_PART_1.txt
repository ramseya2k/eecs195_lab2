For Part 1: 
To run the code
1) roslaunch turtlebot3_gazebo turtlebot3_empty_world.launch
2) rosrun TurtleBot Motion_Planner_Part1.py 
3) rosrun TurtleBot PID_Controller.py 
4) Enter in the coordinates you'd like to see the robot approach, the theta, and the mode. 
5) Once the robot reaches the goal, the motion planner should prompt again for input.
